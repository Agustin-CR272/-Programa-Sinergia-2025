Introducción

HTML es el lenguaje principal para crear páginas web.

Se compone de elementos llamados etiquetas.

<!DOCTYPE html> <html></html>
  • Lenguaje de marcado
  • Base del desarrollo web
Etiquetas Básicas

Algunas etiquetas básicas incluyen:

<h1>, <p>, <div>

<h1>Título</h1> <p>Texto</p>
  • Títulos
  • Párrafos
  • Contenedores
Listas en HTML

HTML permite crear listas ordenadas y no ordenadas.

Estas son útiles para agrupar contenido.

<ul><li>Item</li></ul> <ol><li>Paso 1</li></ol>
  • Listas con viñetas
  • Listas numeradas
Imágenes y Enlaces

HTML permite insertar imágenes y enlaces.

Esto mejora la experiencia del usuario.

<img src="imagen.jpg" alt="Descripción"> <a href="https://ejemplo.com">Enlace</a>
  • Enlaces internos
  • Imágenes representativas
Buenas Practicas

Es recomendable escribir código limpio y semántico.

Usa indentación y comentarios.

<!-- Comentario --> <section></section>
  • Comentarios útiles
  • Etiquetas semánticas
